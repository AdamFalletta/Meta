Pillar Two: Foundry

02. RevV-X Chiplets & NEO Processing Units

The **RevV-X Chiplet** represents the core of AFI’s hardware strategy. Built on a 5nm process node, it is designed for ultra-low latency, mission-critical processing required by high-speed interactive systems like NeoGear and Knockout!. Its modular design allows it to be scaled from a tiny embedded sensor (NPUs) to a full-scale cloud accelerator.

NPUs (Neo Processing Units)

Embedded sensors and edge computing for real-time data ingestion and pre-processing, reducing cloud load by 60\%.

OCX1 \& ACA1 Modules

Core Microcontroller and SPI-linked devices acting as the communication bridge between physical sensor clusters and the MetaOS cloud infrastructure.
