Overview

This page presents AFI-branded deployment products that map the environment matrix: Local → Global and On-Premise → Off-Premise, each as a product tier with a short tagline and a practical example.

01

AFI BoothCast™

Pop-up multiplayer. Anywhere. No cloud required.

Inside a small event venue, an AFI BoothCast™ unit hosts Knockout! on a vertical 55” display connected to a local Wi‑Fi access point. Attendees scan a QR code to join matches via their phones — no internet required. The BoothCast Server Node (a local Raspberry Pi or micro‑PC) broadcasts the game state only within the building, allowing hundreds of players to compete with zero lag. After the event, the entire unit can be unplugged and redeployed in minutes.

02

AFI VaultCore™

Offline intelligence. Total control.

In AFI’s prototype lab, an AFI VaultCore™ cluster runs early builds of CloudPress and MetaOS inside a secure network room. Access is locked down — only engineers on the same subnet or VPN tunnel can reach it. Disconnected from the public internet, VaultCore guarantees zero leaks and top performance for firmware builds and ACA1 testing. Logs, backups, and build assets stay inside the VaultCore Array, under physical lock and digital key.

03

AFI MetroNet™

Regional reach. Local latency.

AFI partners with a local telecom to deploy AFI MetroNet™, a city‑wide leaderboard hub for arcade tournaments. Hosted at the ISP’s datacenter, it’s accessible only to users in that metro region. Residents connect from home or mobile, joining brackets or downloading updates — but traffic never leaves the city. MetroNet delivers low‑latency play without the global cloud footprint.

04

AFI EdgeBox™

Private edge power for serious compute.

AFI rents a half‑rack at a nearby colocation facility to run AFI EdgeBox™, a private edge cluster for regional AI caching and video rendering. Not publicly reachable, it connects to AFI’s internal network via VPN or dedicated fiber. EdgeBox gives local engineers fast, secure access to high‑performance compute without relying on third‑party clouds.

05

AFI WorldHub™

Global play, powered from home base.

The AFI WorldHub™ server powers Space Rage multiplayer directly from HQ. Players worldwide connect via AFI’s public IP to the main hub — a cooled, 24/7 on‑site rack with fiber uplink. It grants AFI total control over global hosting and instant patch access, embodying true homegrown global reach.

06

AFI SourceNet™

A private global network for creators.

The AFI SourceNet™ datacenter serves authenticated engineers worldwide. Teams in Japan, Brazil, and the U.S. connect via encrypted VPN to access AFI’s internal Git, build systems, and documentation vault. Fully managed in‑house, SourceNet provides global accessibility with full data sovereignty.

07

AFI CloudPress™

Publish to the planet.

AFI CloudPress™ runs across AWS and Azure worldwide, powering website creation and publishing for users everywhere. Anyone can sign up, publish, and access content with global uptime and auto‑scaling under heavy load. CloudPress orchestrates across regions, routing users to the nearest node for lightning‑fast delivery.

08

AFI CloudCore™

The private backbone of the AFI ecosystem.

AFI CloudCore™ is AFI’s global private backbone — a distributed mesh spanning continents, hosted across partner datacenters. No public endpoints, only encrypted interlinks. CloudCore runs enterprise AI, simulation, and inter‑game data exchange — the silent nervous system connecting every AFI system worldwide.
