Core Concepts: Modular Design

Metablocks

**Self-contained, functional circuit blocks** with standardized interfaces. They are the core building blocks, enabling **plug-and-play compatibility** for rapid prototyping.

  • Example: ESP32 WiFi Controller, 5V DC-DC Converter.
  • Benefit: Global component reuse via the Marketplace.

Plugins & Extensibility

Dynamically **extend the platform's functionality** for specialized tasks, running as sandboxed Docker apps in the cloud or locally.

  • Simulation: SPICE, FPGA, Thermal Analysis.
  • Deployment: Auto-Gerber file generation, CI/CD pipelines.

Cloud-Native Collaboration: Real-time editing and commenting, achieving Google Docs-like productivity for complex circuit designs.

The Four-Layer Architecture

A robust, containerized microservices infrastructure built for scale and real-time performance.

User Layer

Web client (React/Next.js) and optional desktop IDE. Features include Drag-and-Drop Editor and versioning.

Application Layer

The core logic: Project Manager, Metablock Engine, Plugin Manager, and the WebSockets-based Collaboration Engine.

Data Layer

PostgreSQL/CockroachDB for metadata. S3/GCS Object Storage for large artifacts like Gerbers and simulation results.

Infrastructure

Scalable microservices via Docker/Kubernetes to handle heavy load from simulations and CI/CD pipelines.

Full Lifecycle Management
